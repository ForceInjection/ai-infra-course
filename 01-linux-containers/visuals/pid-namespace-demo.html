模块 1 — Linux 基础与容器技术入门  |  PPT 第 14 页  |  Host 视角 vs Container 视角

容器进程 普通进程 内核线程 PID 映射
💻 Host (宿主机)能看到所有进程
Host 视角：PID 是全局唯一的。看到所有 namespace 的进程，包括容器内的进程（用真实 PID 显示）。
📦 Container (容器内)只看自己 namespace
Container 视角：PID 从 1 开始重新编号。只能看到自己 namespace 内的进程，看不到宿主机其他进程。同 namespace 的进程用容器内 PID 显示。
↑ Host PID ↔ Container PID 映射 ↑
同一个进程在 Host 和 Container 中有不同的 PID
