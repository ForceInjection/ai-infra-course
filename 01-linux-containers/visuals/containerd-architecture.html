💡 containerd 是什么？

containerd 是业界标准的容器运行时，负责管理容器的完整生命周期。

上游：dockerd、kubelet 等通过 gRPC API 调用 containerd 创建容器
核心：镜像管理 + 容器元数据 + 文件系统快照
下游：为每个容器 fork 独立的 containerd-shim → runc → 容器进程
