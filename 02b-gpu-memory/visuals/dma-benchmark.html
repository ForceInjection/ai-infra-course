DMA (Direct Memory Access) — 为什么 Pinned 比 Pageable 快 2.7×？

当 CPU 发起 cudaMemcpy(dst, src, size, cudaMemcpyHostToDevice) 时，数据如何从系统 RAM 到达 GPU 显存？

答案取决于内存类型：Pinned Memory（页锁定）可以直接被 GPU DMA 引擎访问，Pageable Memory（可分页）需要操作系统介入做一次额外的 CPU 拷贝。
