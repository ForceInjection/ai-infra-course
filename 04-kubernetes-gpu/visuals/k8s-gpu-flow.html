💡 K8s GPU 调度全链路

一个 GPU Pod 从 kubectl apply 到容器内 nvidia-smi 可见，经过 7 个步骤。
① kubectl apply → ② API Server → ③ Scheduler → ④ kubelet → ⑤ Device Plugin Allocate → ⑥ NVIDIA CTK hook → ⑦ 容器启动

你只写了一句 nvidia.com/gpu: 1，K8s 自动完成剩下所有事。
